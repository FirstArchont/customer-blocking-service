# Проектирование спецификации и модели данных для сервиса управления блокировками клиентов

## Описание
В данном проекте мною была спроектирована спецификация Open API / Swagger для сервиса управления блокировками клиентов, а так же была составлена ERD диаграмма, показывающая связь пользователей и сотрудников с блокировками в базе данных

## Содержание
| Артефакт | Описание |
|----------|----------|
| **Описание спецификации** | Open API / Swagger спецификация в формате YAML |
| **ERD диаграмма** | связь пользователей и сотрудников с блокировками в базе данных |
